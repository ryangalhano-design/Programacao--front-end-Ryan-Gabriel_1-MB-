# 🌱 Agrinho Sustentável

Projeto educativo inspirado no programa Agrinho.

## 🌎 Tema
- Meio ambiente
- Agricultura sustentável
- Relação entre campo e cidade

## 💡 Objetivo
Mostrar como podemos conciliar desenvolvimento urbano e rural sem destruir o planeta.
